Working with JS Libraries
RECAP
  • +Libraries are a file or set of files made up of functions
  • +They typically help speed up development, especially if they 'wrap' other code
Working with JS Libraries
RECAP
  • +You typically link to them with <script> tags in your HTML
  • +You may also see import or include or require
  • +These typically occur when JS files need to include functions from other JS files
Working with JS Libraries
RECAP
  • +Download the library files, typically inside a js or js/libs folder
  • +Having copies of the files locally makes things faster
  • +Most popular libraries are available via Content Delivery Network, or a CDN
  • +Because you're getting these files from the cloud, it will increase load time

Intro to JQuery
What is JQuery?
  • +JQuery is a utility library that makes it easier to code aspects of Javascript
  • +It has a lot of functionality for event handling, interactivity
  • +It makes it easier to target and batch select elements
Intro to JQuery
$(document).ready()
  • +The $ is essentially an alias/shorthand for JQuery()
  • +Some other libraries use the $ as well, so watch out for conflicts 
  •

Intro to JQuery
Adding & Updating Multiple Elements 
  •                   
                    <!-- /exercises/03-jquery-multipledivs.html -->                           
                    $(document).ready(function() {
                      setupDivs();
                      
                      $('#myBtn').click(function() {
                        $('div.custom-div').css("background", getRandomColor());

                    <!-- ---------------------------------------------------- --> 
                    function getRandomColor() {
                      let letters = '0123456789ABCDEF';
                      let color = '#';
                      for (let i = 0; i < 6; i++) {
                        color += letters[Math.floor(Math.random() * 16)];
                      }
                      return color;
                    }
                    
                    <!-- ---------------------------------------------------- --> 
                    function setupDivs() {
                      let markup = '';
                      let numDivs = 48;
                      
                      let singleDivMarkup = '<div class="custom-div"></div>';
                      for (let i = 0; i < numDivs; i++) {
                        markup += singleDivMarkup;
                      }
                      
                      $('body').append(markup);
                    }

Intro to JQuery
Attaching Multiple Events
  • +Notice that we are passing in an object to on() 
  •                   
                    <!-- /exercises/06-jquery-multiple-events.html --> 
                    $(document).ready(function() {
                      setupDivs();
                      
                      // add multiple events
                      // notice that we are passing an object to on()
                      $('.custom-div-big').on({
                        click: function() {
                          $(this).css("background", getRandomColor())
                        },
                        
                        dblclick: function() {
                          $(this).css("background", "black");
                        }
                      })
                    });
                    
                    <!-- ------------------------------------------------------- --> 
                    function getRandomColor() {
                      let letters = '0123456789ABCDEF';
                      let color = '#';
                      for (let i = 0; i < 6; i++) {
                        color += letters[Math.floor(Math.random() * 16)];
                      }
                      return color;
                    }
                    
                    <!-- ------------------------------------------------------- --> 
                    function setupDivs() {
                      let markup = '';
                      let numDivs = 6;
                      
                      let singleDivMarkup = '<div class="custom-div-big"></div>';
                      for (let i = 0; i < numDivs; i++) {
                        markup += singleDivMarkup;
                      }
                      
                      $('body').append(markup);
                    }
